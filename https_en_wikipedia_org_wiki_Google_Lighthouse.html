Google Lighthouse

From Wikipedia, the free encyclopedia
Jump to navigation Jump to search
open-source, automated tool for measuring the quality of web pages

Google Lighthouse is an open-source, automated tool for measuring the quality of web pages. It can be run against any web page, public or requiring authentication. Google Lighthouse audits performance, accessibility and search engine optimization of web pages.[1][2][3] It also includes the ability to test progressive web applications for compliance with standards and best practices. Google Lighthouse is developed by Google and aims to help web developers.
