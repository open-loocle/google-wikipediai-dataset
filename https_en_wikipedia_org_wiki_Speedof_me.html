SpeedOf.Me (Speed of me) is a broadband speed test service which uses pure browser capabilities such as HTML5 and JavaScript. It is designed to measure internet connection speed without Flash or Java, which are currently used by many other speed test websites.(it is actually slower than any other website) It works on iPhone, iPad, Windows Phone, Windows 8, BlackBerry, Android and other hand-held devices as well as desktop computers. SpeedOf.Me utilizes multiple test servers around the world, for speed and reliability, and the server is chosen automatically. [1][2][3][4][5][6][7][8][9]
