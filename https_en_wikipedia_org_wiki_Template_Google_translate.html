Usage[edit]

This template is only supposed to be used on talk pages.
Shortcut
  • {{gtlink}}

Both {{google translate|http://ru.wikipedia.org|Russian Wikipedia's Main page|lang=ru}} and {{google translate|url=http://ru.wikipedia.org|title=Russian Wikipedia's Main page|lang=ru}} will render as https://translate.google.com/translate?sl=ru&u=http://ru.wikipedia.org%7C2=Russian Wikipedia's Main page.

This template should be substituted, though transcluding it should also work.

Parameters[edit]

|url= or first unnamed parameter
Link to the page in non-English language
|title= or second unnamed parameter
Title of the link (preferably translated title of the page to be translated (optional, defaults to "translation")
|lang=
two-letter code for the language of the page to be translated; be sure that this code is supported by Google Translate (optional, but recommended)
