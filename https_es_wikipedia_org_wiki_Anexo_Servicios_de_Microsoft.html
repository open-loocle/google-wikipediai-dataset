Microsoft Web Services es el nombre con el que se conoce un pack de aplicaciones web gratuitas producidas por Microsoft a las que se accede a través de un navegador de Internet. Entre los servicios que ofrecen se encuentran el correo electrónico (Outlook.com), almacenamiento y edición de archivos en línea (OneDrive), gestión de contactos, chats y otros servicios sociales y calendario. De algún modo se puede considerar el sucesor de Windows Live. Se accede a ellas mediante ID de Microsoft, o lo que es lo mismo, una cuenta Microsoft account.

Servicios de búsqueda[editar]

Artículo principal: Bing

Bing (anteriormente Live Search, Windows Live Search y MSN Search) es un buscador web de Microsoft. Incluye la lista de sugerencias de búsqueda en tiempo real y una lista de las búsquedas relacionadas (llamado "Panel de explorador" en el lado izquierdo de los resultados de búsqueda).

Servicios para Desarrolladores[editar]

Artículo principal: Live Connect

Live Connect es una colección de APIs y controles comunes que permiten a los desarrolladores tener un control más completo y datos abiertos y accesibles para las interfaces de programación de aplicaciones (API). Messenger Connect se basa en tecnologías web estándar como ASP.NET, Microsoft Silverlight (en el navegador y el navegador de los modelos hacia fuera), Windows Presentation Foundation (WPF), Adobe Flash, PHPy Java , y está diseñado para funcionar con cualquier tecnología o dispositivo.
