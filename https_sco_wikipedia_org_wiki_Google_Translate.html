Google Translate is a machine translation wabsteid frae Google that can translate text atween different langages.[4] It can translate wirds, sentences, blads and wab pages atween onie mixtur o mair nor 100 supportit langages.[5] It wis launtched on the 28 Aprile 2006 and haes aye been free tae uise.[6]

Google Translate can dui mair nor shaw juist the translate o wirds. It can shaw hou tae say the wird uisin text-tae-speak, and shaw the meanin like a dictionar an aa. It lats the communitie mak chynges or suggestions tui, for example, if the computer made a wrang translate. This is a lot like hou a bodie can pit up an chynge things on wabsteids like Google Maps an Wikimedia Foundation wabsteids.
